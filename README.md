### WanAndroidMVVM
- 该项目基于 [张鸿洋](https://github.com/hongyangAndroid) 大佬的 [玩Android](https://www.wanandroid.com/blog/show/2) 接口
- 项目采用 Kotlin 语言
- 整体采用  MVVM 架构编写
- 使用到的 JetPack组件有 LiveData 、 ViewModel 、 Lifecycles 、 DataBinding 、 Navigation 、 Paging 3.0

### 已有功能

- 首页最新文章
- 搜索
- 每日一问文章列表
- 广场文章列表
- 项目文章列表
- 知识体系
- 网站导航
- 公众号
- 收藏、分享文章
- 代办清单
- 用户详细页
- 积分详细、排名
- 夜间模式

### 计划添加的功能

- 混淆配置
- WebFragment 点击事件
- 主题色选择（夜间模式时无效）
- 。。。。。。

### 待完善功能

- 冷启动闪屏
- 积分排名界面
- 部分功能的实现逻辑
- 主题颜色的统一
- 夜间模式
